# Path to Graduation

Path to Graduation! is a 2D tower defense game developed using the Godot game engine as part of a senior software development project. 
Players defend their academic progress by placing towers to stop waves of coursework-themed enemies.
The game includes increasing difficulty, resource management, and a final boss encounter represented as a final exam. 

## To run the game: 

1.) Open the "Executable" folder. 
2.) Double-Click the "PathToGraduation.exe" executable. 
3.) The game will then launch. 

## Controls:

-Left Mouse Button: Select and place towers
-Spacebar: Start enemy wave

## Instructions

-Select a tower from the right panel
-Place towers on valid tiles
-Press the spacebar to start wave
-Towers will automatically attack enemies in range
-Defeat enemies to gain resources
-Use the Left Mouse Button to select and upgrade towers using resources
-Survive all waves and defeat the final boss to win
-If your health reaches zero, game over

## Structure

- Executable/ contains playable game (.exe and .pck)
- scenes/ include game scenes
- scripts include GDScript files
- assets include sprites, sound files, and other assets
